Top 10 Most beautiful Linux distros

Deepin

Deepin is a Linux distribution based on Debian's stable branch. It features DDE, the Deepin Desktop Environment, built on Qt and available for various distributions like Arch Linux, Fedora, Manjaro and Ubuntu. As of version 15.10 it also uses dde-kwin, a set of patches for KDE Plasma's window manager. This Distro won't be easy for all users as it is based on base Chinese language, so most of the apps wouldn't be comfortble to the users.

UbuntuDDE Remix

UbuntuDDE Remix is a Linux distribution based on Debian branch. It have a modern and beautiful enviroment, built on Deepin and Ubuntu base, This Distro also includes gorgeous theming options and preloaded apps for offline work, learning and games to play. It is designed to feel comfortable and intuitive, making it easy to use even if you have little or no computer experience. It is definitely based on Deepin but its base language is English, so it would be easy for all its users.

Garuda Linux

Garuda is a Linux distribution based on Arch Linux branch. It features BTRFS, and a neon, beautiful and clasic Desktop Themes,Garuda is available available in various distributions like GNOME, XFCE, LXQT-Kwin, QTile, etc. It comes with Linux-zen kernel and ZRAM enabled by default for more performance

Zorin OS

Zorin OS is a personal computer operating system designed and promoted for users new to Linux-based computers. One of its built-in features lets users change the interface to resemble those of Microsoft Windows or macOS. It is easy to use because it supports a wide variety of applications and Windows apps support

Elementary OS

Elementary OS is a Linux distribution based on Ubuntu LTS. It promotes itself as a fast, open, and privacy-respecting replacement to macOS and Windows and has a pay-what-you-want model. The operating system, the desktop environment, and accompanying applications are developed and maintained by Elementary, Inc.

Fedora Linux

Fedora is a Linux distribution developed by the community-supported Fedora Project which is sponsored primarily by Red Hat, a subsidiary of IBM, with additional support from other companies. Fedora contains software distributed under various free and open-source licenses and aims to be on the leading edge of free technologies. Fedora is the upstream source of the commercial Red Hat Enterprise Linux distribution, and subsequently CentOS as well.

Solus

Solus (previously known as Evolve OS) is an independently developed operating system for the x86-64 architecture based on the Linux kernel and a choice of the homegrown Budgie desktop environment, GNOME, MATE or KDE Plasma as the desktop environment. Its package manager, eopkg, is based on the PiSi package management system from Pardus Linux. The developers of Solus have stated that Solus is intended exclusively for use on personal computers and will not include software that is only useful in enterprise or server environments.

Feren OS

Feren OS is a free and open-source operating system with a pseudo-rolling-release nature. It is designed to be stable, more powerful, and more secure. With a new yet familiar user experience now included. powered by the KDE Plasma Desktop, Feren OS is a relatively lightweight alternative to most Operating Systems in today's modern world. Designed for modern hardware, Feren OS uses less memory than some of its competitors, and its own past versions.

Nitrux

Nitrux is a Linux desktop distribution directly based on Debian. It uses the Calamares installer and includes NX Desktop and NX Firewall on top of the KDE Plasma 5 desktop environment and MauiKit Applications. It is a simple and minimal yet powerful operating system

eXtern OS (Experimental)

eXtern OS is a custom Linux desktop distribution based on node.js. It uses basic web development and web coding to develop apps like basic web applications. It have extremely beautiful desktop enviroment with a lot of customizations which makes user easy to switch to it. This project is currently in its beta testing and may have bugs in it.
